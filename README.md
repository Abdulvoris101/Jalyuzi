## Jalyuzi web app
Jalyuzi.uz - created for selling products in internet, and this web app have many features. 

 

 **Used Tech stacks:**
   
- Nuxt 3  for frontend
- Django for backend
- Telegram API
- Instagram API
- Slide.js
- Django Rest Framework
- Bootstrap Icons
- Jazzmin Admin panel

**Web App** -  https://jalyuzi.com
